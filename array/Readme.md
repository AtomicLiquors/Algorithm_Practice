### 2차원 배열 돌리기
시계방향 90도 기준
- r행은 N-1-r열이 된다.
- c열은 r행이 된다.
```java
for(int r=0; r<N; r++){
	for(int c=0; c<N; c++) tmp_arr[c][N-1-r] = arr[r][c];
}
```
